# Eye Movement

### Project Description
Eye Movement is an example of combining Javascript with CSS to make the eyes(balls) follow the cursor on the screen. 

### How to Run
Download the files into a single top level directory and open the index HTML file in a browser. 

### Roadmap of future improvements
Future improvements include making the eyeballs into squares or reversing the direction of the eyes to evade looking at the cursor.
